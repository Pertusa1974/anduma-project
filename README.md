# Andüma! — Discover Piedmont

The most comprehensive multilingual travel guide to Piedmont, Italy.

**[anduma.io](https://anduma.io)**

8,200+ restaurants, hotels, events, experiences, and points of interest
across all 8 provinces of Piedmont.

Available in 9 languages: Italian, English, French, German, Spanish,
Portuguese, Dutch, and Japanese.

## Features

- Interactive trip planner
- Live events and sagre calendar
- Multilingual AI-generated descriptions
- SSR pages with structured data for Google
- Progressive Web App — works offline

Built by Marco Cucco in Turin, Piedmont.
